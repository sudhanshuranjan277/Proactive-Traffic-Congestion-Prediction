# TransGTR - Transferable Graph Structure Learning for Traffic Forecasting

## Overview
This folder contains the exact implementation of the paper:

**Transferable Graph Structure Learning for Graph-based Traffic Forecasting Across Cities (KDD 2023)**

## Objective
Reproduce the original paper before proposing any improvements.
